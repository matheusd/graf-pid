# graf-pid

Demo/tutorial para construção de PIDs para controle de motores/máquinas/CNC.

Apenas um demo que escrevi para ensinar um pouco sobre como funcionam PIDs 
em um contexto de máquinas CNC, usando programação.

Nele faço passo a passo um possível caminho sobre como "inventar" um PID, 
utilizando um pouco de intuição de programação.

Usa JQuery, Ace (editor Javascript) e canvas html5 para gráficos.


## English

Demo/tutorial on how to build a PID controller for CNC machines.

This is a simple demo I wrote to teach someone about how PIDs work in the
context of CNC machines, using programming.

I wrote step-by-step instructions on how a programmer might "invent" PID loops
using a bit of intuition on how an automated machine should work.

Uses JQuery, Ace (Javascript Editor) and HTML5 canvas for graphics.
